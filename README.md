# ImpactoFit

Aplicación web móvil para crear y almacenar hojas de actividades físicas que funcionan como clases, turnos, lecciones, disciplinas, talleres o módulos.

## Características

- Lista de bloques de actividades físicas.
- Selección de bloques para construir hojas de entrenamiento.
- Botón de tiempo junto a cada actividad (5, 10, 15, 20 minutos).
- Creación de actividades propias.
- Edición de actividades en un modal amigable.
- Guardado automático de cambios cuando se edita una hoja cargada.
- Vista de duración total estimada de la hoja.
- Filtrado de hojas guardadas por categoría.
- Ordenamiento de hojas guardadas por fecha o duración total.
- Guardado de hojas en el navegador con localStorage.
- Exportar e importar hojas en formato JSON.
- Botones con iconos Google Fonts para una experiencia visual más clara.
- Interfaz responsive para celular.

## Cómo usar

1. Abre `index.html` en un navegador moderno.
2. Busca actividades o explora la lista.
3. Selecciona un tiempo y agrega las actividades a la hoja actual.
4. Ingresa un nombre para la hoja (clase, turno, lección, disciplina, taller o módulo).
5. Presiona "Guardar hoja".
6. Abre, edita o elimina hojas guardadas desde el panel inferior.
7. Usa "Exportar hojas" para descargar tu archivo JSON.
8. Usa "Importar hojas" para subir un archivo JSON y recuperar hojas.
